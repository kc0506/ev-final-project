%================= ROW 1 : Motivation | Method ================
\begin{columns}[t,totalwidth=\textwidth]
\begin{column}{0.49\textwidth}
\begin{block}{Motivation}
Traditional system identification treats an object as a \textbf{homogeneous}
material at \textbf{rest}, moving uniformly in one direction. We pose a harder
problem: recover \textbf{complex initial conditions on a non-uniform material}.
We address four limitations of prior methods:
\begin{itemize}
  \item \textbf{Rest-state assumption.} How can we learn an object
        \emph{released from a deformed state} at $t{=}0$?
  \item \textbf{Scalar only.} Can we learn a \emph{field} to model richer setups?
  \item \textbf{Unstable identification.} When is the system actually
        identifiable?
  \item \textbf{Plain $L_2$ losses.} They are not good enough for oscillations.
        We propose a \textbf{spectral loss}, and jointly train initial velocity
        and material while analysing which one dominates.
\end{itemize}
\end{block}
\end{column}

\begin{column}{0.49\textwidth}
\begin{block}{Method}
\centering

  \draw[->,line width=1.2pt,pblue] (loss.north) to[bend left=22]
        node[above,font=\scriptsize,pblue]{$\nabla$ update $v_0,F_0,E$} (mpm.north-|sc);
\end{tikzpicture}

\vspace{2mm}\raggedright
Field trainables use a grid or an MLP queried at the particle position.
\textbf{Learning $F_0$.} The rest state $X$ is \emph{unknown}; only the deformed
state $x_0$ is observed. We therefore express the displacement as a function of
the \emph{known} $x_0$, so $X=x_0-\hat u_0(x_0)$ and
\[
  F_0=\frac{\partial x_0}{\partial X}
     =\Big(\tfrac{\partial X}{\partial x_0}\Big)^{-1}
     =\big(I-\nabla_{x_0}\hat u_0\big)^{-1}.
\]
An MLP parametrises $\hat u_0(x_0)$; $\nabla_{x_0}\hat u_0$ comes from autograd on
the known $x_0$, then we invert.
\end{block}
\end{column}
\end{columns}

\vspace{2mm}\raggedright
\textbf{Conclusion.} $E$ (and $\nu$) controls the \textbf{frequency} of motion,
while $v_0/F_0$ only set its \textbf{amplitude}. The excitation type therefore
decides which channel carries each parameter.
\end{block}
\end{column}

\begin{column}{0.49\textwidth}
\begin{block}{Analysis: a spectral loss for oscillations}
When the frequency mismatch is large, a time-domain $L_2$ loss between two
oscillations is \textbf{bounded} (distance saturates at $2A$), giving a flat
plateau and local minima. A \textbf{spectral} loss stays discriminative.

\vspace{2mm}\centering
\includegraphics[width=0.49\linewidth]{figures/f0_spectra.png}\hfill
\includegraphics[width=0.49\linewidth]{figures/f0_period.png}\\[1mm]
{\footnotesize Left: spectra across $E$. Right: period $\propto 1/\sqrt{E}$,
until it hits a Nyquist wall (an aliasing false basin at high $E$).}
\end{block}
\end{column}

%================= ROW 5 : Augmented learning | Takeaways =====
\begin{columns}[t,totalwidth=\textwidth]
\begin{column}{0.49\textwidth}
\begin{block}{Augmenting a non-identifiable scenario}
When the excited axis points toward the camera, the initial velocity $v_y$ is
hard to learn from a single view: $(E,v_0)$ collapses
($E{\to}\svBestE$, $-\svErrE\%$). \textbf{Adding a second view resolves it}
($E$ \mvErrE\%, $v_0$ \mvErrVz\%).

\end{block}

\begin{column}{0.49\textwidth}
\begin{block}{Takeaways}
\begin{enumerate}
  \item We recover \textbf{complex initial conditions} (released from a deformed
        state, $F_0$) and \textbf{non-uniform fields} ($v_0$, $E$), beyond the
        rest-state homogeneous assumption.
  \item A \textbf{spectral loss} reads the frequency channel that a time-domain
        $L_2$ loss cannot, with a clear Nyquist limit.
  \item $E$ sets frequency, $v_0/F_0$ set amplitude: excitation design governs
        identifiability.
  \item \textbf{Future:} amortise a scene-conditioned physics prior and sample
        plausible physics (VSD $+$ reconstruction guidance).
\end{enumerate}
\end{block}
